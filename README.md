zulu-time
=========
An ISO 8601 extended & IETF compliant UTC Zulu notation _(Z)_ timestamp generator.

Install
-------
```sh
$ npm install --save zulu-time
```

Usage
-----
```js
const zuluTime = require('zulu-time');

zuluTime(); // > 2017-08-14T00:17:18Z
```

Testing
-------
```sh
$ npm test
```
